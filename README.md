# Supply Chain Analytics — Power BI Dashboard

Interactive Power BI dashboard analyzing 180,000+ supply chain orders 
across multiple regions to identify delivery bottlenecks and profitability trends.

## Dashboard Pages
- Executive Summary — KPI cards, profitability distribution, delay by region
- Delay Analysis — Bottleneck detection across shipping mode, segment, department
- Profitability Analysis — Profit vs loss breakdown by region and shipping mode
- Time Trends — Delay and profit patterns by month and day of week

## Tech Stack
- Power BI Desktop
- DAX Measures
- Power Query

## Dataset
DataCo Supply Chain Dataset — 180,000+ orders
